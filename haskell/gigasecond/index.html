Gigasecond

Write a program that will calculate the date that someone turned or will celebrate their 1 Gs anniversary.

A gigasecond is one billion (\(10^9\)) seconds.

Source

Chapter 9 in Chris Pine's online Learn to Program tutorial. view source

Solution

First, I imported the type, UTCTime, and function, addUTCTime, from Data.Time.Clock. 

import Data.Time.Clock (UTCTime, addUTCTime)

Then the task at hand is as simple as adding 1000000000 seconds to a given UTCTime. 

fromDay :: UTCTime -> UTCTime
fromDay = addUTCTime 1000000000
